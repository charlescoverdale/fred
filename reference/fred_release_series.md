# List series in a release

Returns all series belonging to a given release.

## Usage

``` r
fred_release_series(release_id)
```

## Arguments

- release_id:

  Integer. The release ID.

## Value

A data frame of series metadata.

## Examples

``` r
if (FALSE) { # \dontrun{
# GDP release
fred_release_series(53)
} # }
```
